<h3>Silver Screen</h3>
ㄴ 장면을 기억하고, 장소를 기록하는 나만의 영화 티켓

<br clear="left" />
<br/>

- 🎞️ **Recommend & Wish** : 영화 추천 및 위시리스트 저장
- 🎟️ **My Ticket** : 기억에 남는 장면, 영화 제목, 별점, 명대사 또는 한 줄 소감, 관람일, 관람 장소를 직접 입력하여 티켓 아카이빙
- 🗺️ **Movies Map** : 영화 관람 장소를 지도에 마커로 표시
